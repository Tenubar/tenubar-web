## Overview
**TenuSpeak** is a real-time subtitle tool built for livestreamers. It pairs a desktop overlay with a simple web experience so creators can run live captions without interrupting their stream.

---

## Key Features
1. **Live Subtitles:** Real-time captions designed for streaming overlays.
2. **Desktop Overlay:** Lightweight app that stays on top during broadcasts.
3. **Web Dashboard:** Manage your account, settings, and updates in one place.
